# Omega

Premium dark admin dashboard built with React.

**Live URL:** https://omega-dashboard.vercel.app  
**GitHub:** https://github.com/username/omega

## Tech Stack

- React 18
- Vite
- React Router v6
- TanStack Query v5
- Zustand
- Recharts
- Framer Motion
- Tailwind CSS v4

## Features

- [x] Dark premium UI with custom design system
- [x] Responsive layout with collapsible sidebar
- [x] Product management with table/grid views
- [x] Real-time search with debounce
- [x] URL-synced filters, sorting, and pagination
- [x] Column customizer with drag-and-drop reorder
- [x] Animated stat cards with live counter
- [x] Interactive data charts (Recharts)
- [x] Product detail with image carousel
- [x] 30s polling with flash update indicators
- [x] Framer Motion page transitions
- [x] Skeleton loading states
- [x] Error boundary with retry
- [x] 404 handling

## Local Setup

```bash
npm install
npm run dev
```

The app runs at `http://localhost:5173`.
